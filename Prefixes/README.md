# The prefix "re": The Ultimate Comparison Guide 

The prefix "re" means "again. This comprehensive guide breaks down the rules of adding the prefix "re" to a word.

---

## 🔠 1. Generally, don't add or remove any letters when adding the prefix "re", like:
*   **Re-** + **Write** $\rightarrow$ **Rewrite** 
    *   *Example:* If I could **rewrite** the script I would change one thing.
*   **Re-** + **Do** $\rightarrow$ **Redo**
    *   *Example:* These new measurements mean that I'll have to **redo** the calculations.

---

## 📚 2. Using a hyphen for semantic meaning
If the word can change the semantic meaning if omitting it would result a different word with a different meaning, hyphenate it.
*   **Re** + **Cover** $\rightarrow$ **Recover**
    *   *Example:* A police officer was **recovering** in hospital last night after being stabbed.
*   **Re-** + **Cover** $\rightarrow$ **Re-cover**
    *   *Example:* he armchair was worn out, so we decided to **re-cover** it.
 
---

## 📖 3. Using a hyphen if the base word after the word ends with "e"
If the base word starts with "e" and attaches the prefix before the word, put a hyphen. In British, Canadian and Australian English, this rule is necessary whereas American English tends to omit it.
*   **Re-** + **Examine** $\rightarrow$ **Re-examine** (or **reexamine** in American English)
    *   *Example:* All the evidence needs to be **re-examined**. (or **reexamined** in American English)
*   **Re-** + **Enter** $\rightarrow$ **Re-enter** (or **reenter** in American English)
    *   *Example:* Adams decided to **re-enter** politics after the war. (or **reenter** in American English)
